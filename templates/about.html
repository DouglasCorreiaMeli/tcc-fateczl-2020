  • priority_highO que é este projeto

    Este projeto é uma implementação prática de um trabalho de graduação de curso sobre anállise sentimental em redes sociais. O sistema realiza a análise sentimental a partir de um data mining na rede social Twitter aplicando conceitos de Machine Learning e Natural Language Processing.
